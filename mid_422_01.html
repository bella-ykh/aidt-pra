얼굴을 인식하는 로봇 강아지

이미지 분류 모델에 적용하여 주인의 얼굴을 인식하는 로봇 강아지 프로그램을 작성해 보자
로봇 강아지가 나를 알아보고 인사해 주면 좋겠구나.

1해결 방법 생각하기

문제 상황에 맞는 해결 방법과 필요한 데이터 유형을 생각해 보자.
• 해결 방법
할아버지 얼굴 이미지를 학습시켜 할아버지 이미지가 입력되면 로봇 강아지가 인사하도록 프로그램 작성하기
• 데이터 유형
이미지 데이터

2인공지능 모델 만들기

문제를 해결하는 데 적합한 인공지능 모델을 만들어 보자.

①엔트리의 ‘인공지능’ 블록 카테고리의 ‘인공지능 모델 학습하기’를 선택한다.

②‘학습할 모델 선택하기’에서 ‘새로 만들기’의 ‘분류: 이미지’를 선택하고 ‘학습하기’를 클릭한다.

③모델 이름을 ‘주인 얼굴 인식’, 클래스 이름을 각각 ‘주인’, ‘다른 사람’으로 입력한다.

Tip
클래스는 이미지가 어떤 카테고리에 속하는지를 나타내는 것으로, 레이블이라고도 한다. ‘클래스 추가하기’ 버튼을 클릭해서 클래스를 추가할 수 있다.

3인공지능 모델 학습하기

각 클래스에 학습 데이터를 입력하여 모델을 학습시키고 새로운 데이터로 결과를 확인해 보자.
  • ①각 클래스에 해당하는 이미지를 업로드하거나 컴퓨터 카메라로 촬영하여 학습 데이터를 5개 이상 입력한다.
  • ②학습 데이터를 입력한 후, ‘학습하기’를 클릭하여 학습을 진행한다.
  • ③모델 학습이 잘 이루어졌는지 새로운 데이터를 업로드하거나 촬영하여 결과를 확인한다.
  • ④결과를 확인하여 ‘주인’과 ‘다른 사람’ 얼굴을 잘 분류하면 ‘적용하기’를 클릭하고, 그렇지 않다면 해당 클래스의 이미지 학습 데이터를 추가하여 학습시킨다.

4인공지능 모델 활용 프로그램 작성하기

이미지를 인식하는 ‘인공지능 로봇 강아지’ 프로그램을 작성해 보자.
