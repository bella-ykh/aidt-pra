입력한 색에 따라 다른 모양 출력하기(엔트리)

응원 단장 지원이는 다른 반의 플래카드를 보고, 응원 도구를 준비하지 못했다는 것을 깨달았다. 난처해하는 지원이를 위해 눈에 띄는 플래카드 프로그램을 작성해 보자.

1시스템 구성

프로그램의 실행 화면을 구성해 보자.
프로그램의 실행 화면 구성 요소
구분 이미지 설명
입력 큰별(노랑) 오브젝트
누르면 하트 모양을 표시
큰별(빨강) 오브젝트
누르면 위쪽 화살표 모양을 표시
큰별(파랑) 오브젝트
누르면 응원 문구를 표시
출력 LED 디스플레이(구동기)
응원 메시지를 출력하는 하드웨어

2프로그래밍 준비

  • ①USB 케이블을 이용하여 마이크로비트를 컴퓨터와 연결한다.
  • ②하드웨어를 연결하여 마이크로비트 관련 블록을 사용할 수 있도록 한다.

3프로그래밍

①화면 구성하기

  • [+오브젝트 추가하기] 버튼을 클릭한 후 인터페이스 탭에서 ‘큰별(노랑)‘, ‘큰별(빨강)’, ’큰별(파랑)’ 오브젝트를 화면에 추가한다.
  • 오브젝트 속성 창에서 오브젝트의 위치를 설정한다.
    ※ 오브젝트의 위치
    오브젝트의 위치
    오브젝트 축 값
    큰별(노랑) X -130.0
    Y 0.0
    큰별(빨강) X 0.0
    Y 0.0
    큰별(파랑) X 130.0
    Y 0.0
  • [+오브젝트 추가하기] 버튼을 눌러 ‘우주(2)’를 배경으로 설정한다.

②사용할 명령어 블록 알아보기

사용할 명령어 블록 알아보기
명령어 카테고리 블록 이미지 설명
오브젝트를 누르면 아래에 연결된 블록들을 실행한다.
LED에 미리 설정되어 있는 모양을 나타낸다.
블록에서 선택한 LED를 선택한 밝기로 켠다.
입력한 문자열을 LED에 차례대로 나타낸다.

③프로그램 작성하기

프로그래밍 작성하기
오브젝트 명령 블럭

4결과 확인

시작하기 버튼을 눌러 프로그램을 실행하여 실행 결과를 확인해 보자.

5평가

프로그램을 실행한 후 다음 항목을 확인해 보자.
평가 항목
평가 항목 예 아니요

빨간색 큰별 오브젝트를 누르면 마이크로비트 LED에 위쪽 화살표 모양이 출력되는가?
파란색 큰별 오브젝트를 클릭하면 마이크로비트 LED에 Victroy!라는 몬구가 순서대로 출력되는가?
