반티 투표하기

체육대회에 입을 반티를 결정하는 문제를 해결하기 연산자를 활용하여 ‘반티 선호도 조사’ 프로그램의 두 번째 장면(투표하기)인 ‘반티 투표하기’ 프로그램을 작성해 보자.

1문제 이해 및 분석

반티별 점수 합계를 구한 후, 모두 투표하였을 때 ‘결과보기’ 장면으로 이동한다.

프로그램 결과 미리보기

2알고리즘 설계

①‘반티 투표하기’ 단계를 위해 설계한 순서도를 확인해 보자.

②문제 해결을 위해 오브젝트가 구현해야 할 기능을 생각해보자.

오브젝트가 구현해야 할 기능
반티 오브젝트 이미지 초기화 실행 화면과 리스트 초기 설정을 한다.
학생 수 입력 전체 학생 수를 입력받아 총 인원 변수에 저장한다.
반티 후보 점수 입력 오브젝트를 클릭하였을 때 반티 별로 점수를 입력받는다.
모든 사람이 투 표하였을 때 ‘결과보기’ 장면으로 이동한다.
Tip
장면을 추가하고 장면 이름을 ‘결과보기’로 변경한다.

3프로그래밍

①준비하기

프로그래밍 준비하기
오브젝트
변수
  • 총 인원
  • 투표한 사람
  • n (리스트의 항목 번호 )
리스트
  • 반티 리스트
  • 반티 총점 (반티별 누적 점수를 저장)
Tip
반티 리스트를 제외한 나머지 변수와 리스트는 실행 화면에서 보이지 않게 숨긴다.

②프로그램 작성하기

프로그래밍 준비하기
반티 총점 초기화
학생 수 입력 및 변수 보이기
반티별 합계 점수 구하기
'결과보기' 장면으로 이동

4결과 확인 및 디버깅

프로그램을 실행하여 실행 결과를 확인해 보자.
Tip
프로그램을 실행할 때, ‘등록하기’ 장면을 선택한 후 ‘시작하기‘ 버튼을 클릭한다.
