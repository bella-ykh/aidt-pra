학생 정보 예시
속성 내용
속성
  • 영화 제목(title)
  • 장르(genre)
  • 평점 리스트(ratings)
메소드
  • add_rating(rating) : 특정 평점(rating)을 추가. 단, 0~10점 범위만 허용한다.
  • get_average_rating(): 모든 평점의 평균을 계산하여 반환한다.
  • display_info(): 영화의 제목, 장르, 평균 평점을 출력한다.

1. 문제 이해 및 분석

문제 이해를 위해 문제 상태를 정의해 보자.
문제 이해를 위해 문제 상태 정의
상태 정의
초기 상태 Movie클래스, 점수는 빈 상태
목표 상태 영화의 제목과 장르를 저장하고, 평점을 추가할 수 있다.
추가된 평점을 이용해 평균 평점을 계산한다.
영화 정보를 출력한다.
여러 영화 중 가장 높은 평균 평점을 가진 영화를 찾는다.
수행 과정
Movie 클래스 생성
__init__() 메소드를 사용하여 영화의 제목(title), 장르(genre), 평점 리스트(ratings)를 초기화한다.
평점 추가
add_rating(rating) 메소드를 이용하여 평점을 추가한다.
평점이 0~10 사이가 아닐 경우 오류 메시지를 출력한다.
평균 평점 계산
get_average_rating() 메소드를 호출하여 ratings 리스트에 저장된 모든 평점의 평균을 계산한다.
평점이 없으면 0을 반환한다.
영화 정보 출력
display_info() 메소드를 호출하여 영화의 제목, 장르, 평균 평점을 출력한다.
영화 리스트 생성 및 평점 추가
Movie 객체를 여러 개 생성하고, 각각의 영화에 평점을 추가한다.
