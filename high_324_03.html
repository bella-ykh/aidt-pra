If~elif~else문

숫자를 입력받아 양수, 음수, 0을 판단하는 프로그램을 작성하여, if~elif~else문의 동작을 확인해 보자.
실행 결과 예시
숫자를 입력하세요. 55
양수입니다.

1. 프로그래밍

코드를 보고 빈 칸을 채워 프로그램을 완성해 보자.

1 2 3 4 5 
number = int(input('숫자를 입력하세요. '))
if number % 2 == 0:
    print(number,'는 짝수입니다.')
else:
    print(number,'는 홀수입니다.')

2. 결과 확인 및 디버깅

  • ①프로그램을 실행하여 [실행 결과 예시]와 같이 결과가 출력되는지 확인해 보자.
  • ②예시와 같이 결과를 출력했다면 ‘제출하기’ 버튼을 눌러 실습의 결과를 확인해 보자.
