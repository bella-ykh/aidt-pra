다양한 덧셈 프로그램

서로 다른 자료형을 연산하면 어떤 결과가 나올까? 다음 프로그램의 실행 결과를 예상해 보고, 결과를 확인해 보자.

1. 프로그래밍

①코드를 보고 프로그램을 완성하고, 실행 결과를 확인해 보자.

1 2 3 4 5 6 
a, b, c = 30, 30.0, '30'
print(a, b, c) 
print(type(a), type(b), type(c)) 
    
sum1 = a + b 
print(sum1)

②아래 코드를 추가하여 프로그램을 수정해 보자.

8 9 
sum2 = a + c 
print(sum2)

2. 결과 확인 및 디버깅

①프로그램을 실행하여 결과를 확인한 후, 결과가 나온 이유에 대해 생각해 보자.

실행 결과
입력 보류 /// 실제 도구에서 결과값 확인 필요//
Tip!
TypeError : 연산이나 함수에 전달되는 자료형이 잘못된 경우에 발생하는 오류입니다.
(오류 메시지는 위의 실행 결과와 다를 수 있습니다.)

②오류가 발생했다면, 아래와 같은 실행 결과가 나올 수 있도록 프로그램을 수정해보자.

목표 실행 결과
30 30.0 30
<class 'int'> <class 'float'> <class 'str'>
60.0
3030
Tip!
정수와 실수에 대한 + 연산자는 덧셈을 수행하지만 문자열에 대해서는 연결을 수행한다. 따라서 숫자와 문자를 함께 연산하려면 아래와 같은 함수를 사용하여 자료형을 변환해야 한다.
자료형 변환(형 변환, Type Conversion)
학생 정보 예시
함수 설명 예시
str(값) 값을 문자열로 변환 str(30) → ’30’
int(값) 값을 정수로 변환 int(’30’) → 30
float(값) 값을 실수로 변환 float(‘30.5’) → 30.5

3. 결과 제출하기

프로그램을 실행하여 [목표 실행 결과]와 같이 결과를 출력했다면 ‘제출하기’ 버튼을 눌러 실습의 결과를 확인해 보자.
