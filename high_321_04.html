다양한 덧셈 프로그램

서로 다른 자료형을 연산하면 어떤 결과가 나올까? 다음 프로그램의 실행 결과를 예상해 보고, 결과를 확인해 보자. 
a, b, c = 30, 30.0, '30'
print(a, b, c) 
print(type(a), type(b), type(c)) 
    
sum1 = a + b 
print(sum1) 
    
sum2 = a + c 
print(sum2)

Tip!
정수와 실수에 대한 + 연산자는 덧셈을 수행하지만 문자열에 대해서는 연결을 수행한다. 따라서 숫자와 문자를 함께 연산하려면 아래와 같은 함수를 사용하여 자료형을 변환해야 한다.
자료형 변환(형 변환, Type Conversion)
학생 정보 예시
함수 설명 예시
str(값) 값을 문자열로 변환 str(30) → ’30’
int(값) 값을 정수로 변환 int(’30’) → 30
float(값) 값을 실수로 변환 float(‘30.5’) → 30.5
