감정 표현 배지 만들기(메이크코드)

우리는 매일 다양한 감정을 느끼며 살아간다. 이러한 감정을 효과적으로 전달하기 위해 이모티콘을 사용한다. 마이크로비트의 LED 디스플레이를 활용하여 감정 배지 프로그램을 작성해 보자.

1시스템 구성

프로그램의 실행 화면을 구성해 보자.
프로그램의 실행 화면 구성 요소
구분 이미지 설명
입력 A 버튼, B 버튼, 터치 센서
버튼을 누르거나 터치하면 감정을 표시
출력 LED 디스플레이
LED 디스플레이에 감정을 표시

2프로그래밍

①사용할 명령어 블록 알아보기

사용할 명령어 블록 알아보기
명령어 카테고리 블록 이미지 설명
A 버튼을 누르면 감싸고 있는 블록들을 실행한다.
B 버튼을 누르면 감싸고 있는 블록들을 실행한다.
로고를 누르면 감싸고 있는 블록들을 실행한다.
LED 디스플레이에 설정되어 있는 아이콘을 출력한다.

②프로그램 작성하기

프로그래밍 작성하기
코드 명령 블럭
행복함 코드
슬픔 코드
화남 코드

3결과 확인

시뮬레이터의 A 버튼, B 버튼, 로고를 눌러 프로그램을 실행한 후 실행 결과를 확인해 보자.

4평가

프로그램을 실행한 후 다음 항목을 확인해 보자.
평가 항목
평가 항목 예 아니요
A 버튼을 누르면 행복함 이모티콘이 표시되는가?
B 버튼을 누르면 슬픔 이모티콘이 표시되는가?
로고를 누르면 화남 이모티콘이 표시되는가?
