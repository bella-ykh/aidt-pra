정사각형 넓이 계산기

표준입출력 함수를 사용하여 한 변의 길이를 입력받아 정사각형의 넓이를 계산하는 프로그램을 완성해 보자.
실행 결과 예시
정사각형의 넓이를 계산하겠습니다.
정사각형의 한 변의 길이를 입력하세요: 25
변의 길이가 25 인 정사각형의 넓이는 625 입니다.

1. 프로그래밍

표준입출력 함수를 사용하여 프로그램을 완성해 보자.
Hint변수 a에 값을 입력받는 방법
2
a = input('정사각형의 한 변의 길이를 입력하세요: ')
Hint문자열 값을 출력하는 방법
3
print('변의 길이가', a,'인 정사각형의 넓이는', int(a) * int(a),'입니다.')
Tip!
input( ) 함수로 입력받은 값은 문자열로 저장되므로 정수로 변환하고 싶을 때는 정수 변환 함수 int( )를 사용한다.

2. 결과 확인 및 디버깅

  • ①프로그램을 실행하여 [실행 결과 예시]와 같이 결과가 출력되는지 확인해 보자.
  • ②예시와 같이 결과를 출력했다면 ‘제출하기’ 버튼을 눌러 실습의 결과를 확인해 보자.
