리스트로 표현한 그래프

그래프의 노드들이 연결된 정보를 리스트에 저장하는 파이썬 프로그램을 작성해 보자.
다음 그래프를 2차원 리스트로 표현하려고 한다.
리스트에 저장되는 연결 정보는 컴퓨터가 이해하기 쉽도록 0 또는 1로 표현한다. 즉, 노드가 연결돼 있다면 1로 표현하고, 연결되지 않았거나 자신과의 연결돼 있다면 0으로 표현한다.
노드 간 연결성
0번 노드 1번 노드 2번 노드 3번 노드
0번 노드 0 1 0 1
1번 노드 1 0 1 1
2번 노드 0 1 0 0
3번 노드 1 2 0 0
