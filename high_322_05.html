물품 보관함 정보 프로그램

파일입출력 함수를 사용하여 공공 데이터 파일의 내용을 확인하는 파이썬 프로그램을 완성해 보자.
공공 데이터 포털
(https://www.data.go.kr/data/15044234/fileData.do) 에서 내려받은 서울교통공사 물품 보관함 위치 정보 파일(서울교통공사_물품보관함 위치정보_20231031.csv) 을 출력하려고 한다. 파일의 내용은 아래 그림과 같다.
// [물품 보관함 위치 정보 파일] // 이미지 제작???



Tip!
csv는 comma-separated values의 줄임말로, csv 파일은 데이터를 ,(콤마)로 구분하여 표 형태로 저장한 텍스트 파일이다.
빅데이터 분석을 위하여 대용량 데이터를 저장할 때 많이 사용된다.

1프로그래밍

빈 칸을 채워 코드를 완성해 보자.
Hint

csv 파일을 읽기 모드로 여는 방법

with open('서울교통공사_물품보관함 위치정보_20231031.csv', 'r') as file:
Tip!
with 명령어를 사용하면 연(open) 파일을 자동으로 닫아주므로, 'file.close( )' 명령을 생략해도 된다.
Hint

파일의 모든 줄을 읽고 리스트에 담아 lines 변수에 저장하는 방법

lines = file.readlines()
Tip!
split( ): 문자열에 관한 함수로 다음과 같은 형식으로 사용한다.
문자열.split('구분자')
구분자를 기준으로 문자열을 잘라 리스트로 만들어주는데, 만약 구분자를 생략한다면 공백을 기준으로 자른다.
