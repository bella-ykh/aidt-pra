프로그램 구현

[창의·융합 프로젝트] 우리 동네 약국 알림이
엔트리를 사용하여 우리 동네 약국 알림이 프로그램으로 구현해 보자.

1문제 이해

지도를 배경으로 하고 약국의 위치(위도, 경도)를 각 축으로 한 산점도로 표현하여 약국의 위치를 표현하는 프로그램을 만든다.

2데이터 수집

문제 해결에 필요한 약국 위치 데이터를 수집해 보자.
  • ①보건의료빅데이터개방시스템에 접속한 뒤, 공공데이터 탭에서 공공데이터 목록을 클릭한다.
  • ②약국 현황을 검색한다.
  • ③검색 결과에서 전국 병의원 및 약국 현황을 클릭한다.
  • ④전국 병의원 및 약국 현황 파일을 다운로드한다.
Tip
  • 보건의료빅데이터개방시스템 웹 페이지: https://opendata.hira.or.kr
  • 연도별 데이터가 제공되고 있으므로, 최신의 데이터로 다운로드 받는다.

3데이터 전처리

  • ①수집한 데이터 파일을 연다.
  • ②
    필요한 데이터(위도, 경도, 약국 이름, 전화번호, 주소)를 제외하고, 필요하지 않은 데이터를 삭제한다.
    Tip
    이 활동에서는 시도별 데이터가 제공되어 있다. 필요한 지역의 데이터만 전처리하여 사용한다.
  • ③
    약국의 x좌표(경도)와 y좌표(위도) 값에서 기준 위치의 경도, 위도를 뺀 값을 기준으로, 지도 화면 크기(가로 -240~240, 세로 -135~135)에 맞게 숫자를 비례해서 조정한다.
    Tip
    이 과정을 좌표 변환 또는 비율 조정이라고 하며, 실제 지도를 화면에 그릴 때 자주 사용하는 방법이다.

4우리 동네 약국 데이터 추가하기

①데이터 불러오기

‘데이터 분석’ 블록 꾸러미 ➜ ‘테이블 불러오기’ 클릭

②데이터 추가하기

테이블 추가하기 ➜ 테이블 선택 ➜ "월평균 미세 먼지 농도" 선택 ➜ 오른쪽 상단의 추가하기 클릭

5프로그램 작성하기

  • ①
    지도 장면이 시작되면 약국 정보를 표시하기 위해, 테이블의 정보를 이용하여 약국의 개수만큼 동그란 버튼 오브젝트를 복제한다.
    프로그래밍 작성하기
    오브젝트 명령 블럭
  • ②
    약국을 나타내는 '동그란 버튼' 오브젝트가 생성 될 때마다, 저장된 약국의 위치 정보를 이용하여 오브젝트 위치를 이동하고 '표시정보' 변수에 약국 정보를 저장한다.
    프로그래밍 작성하기
    오브젝트 명령 블럭
Tip
우리 동네 약국 데이터 파일을 새롭게 추가했다면, 블록에서 영역의 테이블명을 사용할 파일로 변경한다.
