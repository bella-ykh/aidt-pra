독서록 프로그램

파일입출력 함수를 사용하여 책에 대한 한 줄 평을 텍스트 파일에 저장하는 프로그램을 완성해 보자.
// [저장된 텍스트 파일 예시] // 이미지 제작???

1프로그래밍

빈칸을 채워 ‘bookervier.txt’ 파일에 독서록을 저장하는 프로그램을 완성해보자.
Hint

‘bookervier.txt’ 파일을 추가 모드로 여는 방법

f = open('bookreview.txt', 'a')
Hint

저장할 책 정보를 입력받는 방법

title = input('책 제목: ')
review = input('한 줄 평: ')
Tip!
  • write( ) 함수의 인수로 문자열과 변수가 혼합된 값을 입력하려고 할 때, ‘+’ (더하기) 기호로 값을 연결한다.
  • 화면이나 파일에 한 줄을 쓰고 커서를 다음 줄로 옮기고자 할 때, ‘\n’ (newline) 기호를 사용한다.

2결과 확인 및 디버깅

①프로그램을 실행하여 독서록 내용을 입력해 보자.

②완성한 코드 아래에 다음 코드를 추가하여 텍스트 파일에 저장된 내용을 확인해 보자.

# 파일 읽기
f = open('bookreview.txt', 'r') 
a = f.read() 
print(‘\n’)
print(a) 
f.close()

③텍스트 파일 예시와 같이 결과가 잘 출력된다면 ‘채점 하기’ 버튼을 눌러 실습 결과를 확인해 보자.
