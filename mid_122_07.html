만보기 만들기(메이크코드)

마이크로비트의 가속도 센서를 활용하여 만보기를 만드는 프로그램을 작성해 보자.
문제 상황
최근 걷기 운동을 시작했지만 내가 하루에 얼마나 걸었는지 알 수 없어 아쉽다. 스마트 기기 없이도 걸음 수를 확인할 수 있는 방법이 없을까?

1시스템 구성

프로그램의 실행 화면을 구성해 보자.
프로그램의 실행 화면 구성 요소
구분 이미지 설명
입력 가속도 센서
가속도를 측정하여 움직임을 감지하는 센서
터치 센서
터치 센서(로고)를 누르면 마이크로비트가 반응하는 센서
출력 LED 디스플레이
가속도 센서가 흔들림을 감지하거나 터치 센서를 누르면 LED 디스플레이가 반응

A버튼을 누르면 감싸고 있는 블록들을 실행한다.
흔들리는 동작을 감지했을 때 감싸고 있는 블록들을 실행한다.

3결과 확인

시뮬레이터의 A 버튼, B 버튼, SHAKE, 로고를 눌러 프로그램을 실행한 후 실행 결과를 확인해 보자.
