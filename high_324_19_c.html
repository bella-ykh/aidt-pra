상품 할인 계산기

상품 금액에 따라 할인율을 적용하고, 최종 결제 금액을 계산하는 프로그램을 작성해 보자.
  • ①while문을 이용하여 상품금액을 계속 입력한다.
  • ②더 이상 상품이 없을 때는 0을 입력하면 된다.
  • ③기본 할인율은 2%이다.
  • ④100,000원 이상 300,000원 미만의 금액은 5% 추가 할인이 적용된다.
  • ⑤300,000원 이상의 금액은 10% 추가 할인이 적용된다.
실행 결과 예시
상품 금액을 입력하세요: 80000
할인된 금액: 78400원
상품 금액을 입력하세요: 150000
할인된 금액: 139500원
상품 금액을 입력하세요: 350000
할인된 금액: 308000원
상품 금액을 입력하세요: 0
총 결제 금액: 525900원

1. 문제 이해 및 분석

문제 이해를 위해 문제 상태를 정의해 보자.
문제 이해를 위해 문제 상태 정의
상태 정의
초기 상태 사용자로부터 상품 금액을 반복적으로 입력받기 전의 상태
목표 상태 출력 결과: 입력된 모든 상품 금액에 대해 할인이 적용된 최종 결제 금액(total_price)을 계산하여 출력.
예를 들어, 총 결제 금액: 477000원과 같은 형태로 출력.
중단 조건: 사용자가 0을 입력하면 프로그램 종료.
수행 과정
할인 금액 계산 함수 선언
기본 할인율(discount)을 2%로 설정.
입력된 금액이 100,000원 이상 300,000원 미만일 경우 5% 추가 할인.
입력된 금액이 300,000원 이상일 경우 10% 추가 할인.
discounted_price ← price × (1 - discount) 반환.
프로그램 시작
total_price ← 0으로 설정하여 누적 계산을 준비함.
사용자로부터 입력 반복
상품 금액을 입력받음.
중단 조건 확인
입력된 금액이 0이면 반복을 종료함.
할인 금액 계산 함수 호출
discounted_price ← 함수의 반환값
총 결제 금액 누적
total_price ← total_price + discounted_price
반복 종료 후 결과 출력
"총 결제 금액: total_price원"을 출력
