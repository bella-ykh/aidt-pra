일정 관리 메모장

날짜와 간단한 메모를 키보드로 입력하여 memo.txt 파일에 저장하고 파일의 내용을 출력하는 프로그램을 작성해 보자.
[메모 예시] 25년 1월 7일에 ‘정보 과제 하기‘ 메모를 작성하고자 하는 경우
날짜: 250107
메모 내용: 정보 과제 하기
// [메모 예시] // 이미지 제작???

1문제 이해 및 분석

문제 이해를 위해 문제 상태를 정의해 보자.
문제 이해를 위해 문제 상태 정의
상태 정의
초기 상태 날짜와 할 일 메모 내용이 있는 상태
목표 상태 memo.txt에 키보드로 입력한 메모 내용 저장
수행 과정
  • 날짜와 할 일을 키보드로 입력한다.
  • memo.txt 파일을 만들어서 연다.
  • 날짜와 할 일을 memo.txt파일에 저장하고 저장 메시지를 출력한다.
  • memo.txt.파일을 읽어와서 내용을 출력하여 올바르게 저장되었음을 확인한다.
