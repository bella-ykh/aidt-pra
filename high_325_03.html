메소드 정의

Book 클래스에 포함된 get_years_since_publication( ) 메소드를 통해 해당 책이 출판된 후 몇 년이 지났는지를 계산하는 방법을 살펴보자.

1. 프로그래밍

코드를 보고 빈 칸을 채워 프로그램을 완성해 보자.

1 2 3 4 5 6 7 8 
from datetime import date # date: 날짜를 처리하는 표준 라이브러리 객체

class Book:
    def __init__(self, title, author, publish_year):
        self.title = title
        self.author = author
        self.publish_year = publish_year

    # 책이 출판된지 몇 년이 지났는지를 반환하는 메소드
    def get_years_since_publication(self):
        current_year = date.today().year
        return current_year - self.publish_year

book = Book('데미안', '헤르만 헤세', 1919)
print(book.get_years_since_publication())
