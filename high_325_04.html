도서 대여 프로그램

학급 문고의 대여 프로그램으로 구현할 때 필요한 Book 클래스와 Student 클래스의 코드를 작성해 보자.
  • Book 클래스의 속성은 도서명(title), 저자(author), 대여 여부(is_rented)로 구성된다.
  • Student 클래스의 속성은 학번(id), 이름(name), 대여한 책 리스트로 구성되며, 메소드는 책 대여 메소드(rent_book)를 구현한다.

HintStudent 클래스의 rent_book() 메소드 정의하는 방법

def rent_book(self, book):
    if not book.is_rented:
        book.is_rented = True
        self.rented_books.append(book)
        return f'{self.name}에게 {book.title} 대여함.'
    else:
        return f'{book.title} 대여 중.'
