표준입출력 함수

표준 출력 print( ) 함수와 표준 입력 input( ) 함수를 사용한 프로그램의 실행 결과를 예상해 보고, 결과를 확인해 보자.

1프로그래밍

①표준 출력 print( ) 함수를 사용하여 오른쪽 코드를 완성해 보자.

fruit = '청포도'
print('내가 좋아하는 과일은', fruit, '입니다.')
a = 7
b = 5
print(a,'*', b, '의 계산 결과는', a * b, '입니다.')

②표준 입력 input( ) 함수를 사용하여 오른쪽 코드를 완성해 보자.

age = input('몇 살이세요?')
print('당신은', age, '살이군요.')

2프로그래밍

①프로그램을 실행하여 [실행 결과 예시]와 같이 결과가 출력되는지 확인해 보자.

실행 결과 예시
내가 좋아하는 과일은 청포도 입니다.
7 * 5 의 계산 결과는 35 입니다.
몇 살이세요? 19
당신은 19 살이군요.

②예시와 같이 결과를 출력했다면 ‘채점 하기’ 버튼을 눌러 실습 결과를 확인해 보자.
