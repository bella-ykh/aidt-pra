2차원 딕셔너리

우리 반 학생들의 의복 치수를 2차원 딕셔너리로 저장하는 프로그램을 작성해 보자.
우리 반 학생들의 의복 치수
이름 상의 신발 하의
내빈 95 250 M
미송 85 230 S
민수 100 270 XL
준서 95 235 L
지환 90 250 L

1. 프로그래밍

①코드를 보고 오른쪽 프로그램을 완성해 보자.

class1 = {
    '내빈': {'상의': 95, '신발': 250, '하의': 'M' },
    '미송': {'상의': 85, '신발': 230, '하의': 'S' },
    '민수': {'상의': 100, '신발': 270, '하의': 'XL' },
    '준서': {'상의': 95, '신발': 235, '하의': 'L' },
    '지환': {'상의': 90, '신발': 250, '하의': 'L' }
}

# 2차원 딕셔너리에서 키를 이용하여 값 가져오기
print('미송이의 의복 치수:', class1['미송'])
print('미송이의 상의 치수:', class1['미송']['상의'])

# 2차원 딕셔너리에서 키를 이용하여 값 수정하기
class1['미송']['상의'] = 90
print('미송이의 수정된 의복 치수:', class1['미송'])

2. 결과 확인 및 디버깅

①프로그램을 실행하여 [실행 결과]와 같이 출력되는지 확인해 보자.

실행 결과
미송이의 의복 치수: {'상의': 85, '신발': 230, '하의': 'S'}
미송이의 상의 치수: 85
미송이의 수정된 의복 치수: {'상의': 90, '신발': 230, '하의': 'S'}

②결과를 출력했다면 아래 코드를 실행한 후 결과를 확인해 보자.

print(class1['한준'])

③딕셔너리에 존재하지 않는 키를 출력하려고 하면, 오류가 발생한다. 아래 코드를 추가하여 실행 중 오류가 발생했을 때도 실행되도록 수정해 보자.

Tip!
try except(예외 처리): 실행 중 오류가 발생했을 때 실행을 중단하지 않고 계속 실행하게 한다.
예)
  • try :실행할 코드
  • except : 오류가 발생했을 때 처리할 코드
print('학생의 의류 사이즈를 검색하겠습니다.')
name = input('어떤 학생이 궁금하신가요?')
clothes = input('어떤 사이즈가 궁금하신가요?')
try:
    print(f'{name}의 {clothes} 사이즈는 {class1[name][clothes]}입니다.')
except:
    print('존재하는 키가 아닙니다.')
Tip!
f-string: 문자열의 원하는 위치에 특정한 값을 삽입하는 방법이다. 문자열 앞에 f를 붙이고, 어떤 변수나 표현식을 삽입할 때는 중괄호 안에 넣어 준다.
실행 결과
학생의 의류 사이즈를 검색하겠습니다.
어떤 학생이 궁금하신가요? 한준
어떤 사이즈가 궁금하신가요? 상의
존재하는 키가 아닙니다.

④프로그램이 잘 동작한다면 ‘채점 하기’ 버튼을 눌러 실습 결과를 확인해 보자.
