구구단 방 탈출

구구단 방에 들어왔다. 이 방은 랜덤 구구단 문제를 3문제 이상 맞춰야 탈출할 수 있다. 구구단 방 탈출 프로그램을 작성해보자.

1문제 이해 및 분석

구구단 방 탈출 방법 및 절차를 생각해 보자.
  • ①엔트리봇이 내는 문제를 맞춘다.
  • ②문제를 맞추면 OK 값은 1 증가한다.
  • ③다음 랜덤 구구단 문제를 맞춘다.
  • ④OK값이 3이 되면 구구단 방을 탈출한다.

생각 엔트리봇
  • 방탈출 안내 방법을 알려준다.
  • 초시계를 시작한다.
  • 랜덤 구구단 문제를 3개 맞출때까지(ok=3) 계속해서 문제를 제시한다.
  • 문제의 답을 맞추면, 변수 ok의 값을 1 증가한다.
  • 정답 개수가 2개가 되면 방 탈출을 격려하는 응원의 메시지를 표시한다.
  • 사용자가 입력한 답이 틀리면 오답 메세지를 표시한다.
  • 사용자가 문제를 3개 맞추면 반복을 종료하고, 초시계를 정지한다.
  • ②프로그램에 필요한 변수를 생각해 보자.
  • 단 : 랜덤으로 정한 단 값을 저장
  • 수 : 랜덤으로 정한 수 값을 저장
  • Ok : 맞춘 문제의 개수를 저장
  • ③프로그램에 필요한 연산자를 생각해 보자.
  • 3문제를 맞출때까지 반복
  • 정답인지 판단하여 동작을 선택
