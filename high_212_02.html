전치형 암호화 프로그램

전치형 암호화 방법을 이용해 암호문을 출력하는 프로그램을 실행해 보자.

1. 프로그래밍

코드를 보고 빈 칸을 채워 프로그램을 완성해 보자.

Tip!
학생 정보 예시
input( ) 사용자가 입력한 데이터를 문자열로 변환
print( ) 괄호 안의 내용을 화면에 출력
for 변수명 in range( ) 변수가 괄호 안의 범위 내일 때 명령을 반복
if~else if 조건이 참일 때는 if 명령, 거짓일 때는 else 명령 실행
