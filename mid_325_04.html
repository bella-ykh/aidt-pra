‘점수 합계’ 함수 오류 디버깅

‘반티 선호도 조사’ 프로그램 ‘투표하기’ 장면에서 반티별 점수 합계를 구하는 부분을 함수로 정의해 보았다. 프로그램이 생각대로 동작하지 않는다면 무엇이 문제이고 어떻게 해결할지 생각해 보자.

1함수 정의하고, 코드 수정하기

‘투표하기’ 장면에서 반티 오브젝트를 클릭할 때마다, 반티 후보 별로 점수의 합계를 계산한다. 이 동작을 함수로 정의하고, 정의한 함수를 호출하는 방식으로 코드를 수정하였다.

2테스트하기

프로그램을 실행 해보니, 반티 오브젝트를 클릭했을 때 프로그램에서 반티 점수를 입력받는 코드가 동작하지 않았다. 잘못된 곳을 찾아 보자.

프로그램 결과 미리보기

3디버깅하기

함수를 정의하고 사용하는 과정에서 오류가 발생한 이유를 찾고, 잘못된 곳을 바르게 동작하도록 수정해 보자.

오류가 발생한 코드

Tip
함수 안에 블록을 삽입하여 을 출력해 확인하면서 디버깅해 보자.

4결과 확인 및 디버깅

프로그램을 실행하여 실행 결과를 확인해 보자.
