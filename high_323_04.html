하의 치수 관리 프로그램

1차원 딕셔너리를 사용하여 학생들의 하의 치수 정보를 저장하는 파이썬 프로그램을 작성해 보자.
학생들의 하의 치수 정보
이름 신발
내빈 M
미송 S
민수 XL
준서 L
지환 L

1. 프로그래밍

①표를 보고 빈 칸을 채워 딕셔너리에 하의 치수를 저장하는 코드를 완성해 보자.

Hint딕셔너리에 하의 치수를 저장하는 방법
P_size = {'내빈': 'M', '미송':'S', '민수':'XL', '준서':'L', '지환':'L'}

②준서의 하의 치수를 M으로 수정하고 수정된 값을 확인하는 코드를 완성해 보자.

Hint키를 이용해 딕셔너리의 값을 출력하는 방법
print( P_size['준서'] )
Hint키를 이용해 딕셔너리의 값을 수정하고 출력하는 방법 
P_size['준서'] = 'M'
print( P_size['준서'] )

③기존의 딕셔너리에 재우의 하의 치수(XL)를 추가하는 코드를 완성해 보자.

Hint딕셔너리에 새로운 키:값 쌍을 저장하는 방법 
P_size['재우'] = 'XL'
print(P_size)
