칼로리 데이터와 운동 데이터의 시각화(박스 플롯)

박스 플롯은 데이터의 통계 분포와 이상치를 시각화하기 좋은 도구이다. 코답을 사용하여 박스 플롯을 표현하는 방법을 알아보자.

1. 문제 상황 이해하기

칼로리와 운동 데이터에 있는 속성 중 남녀 성별에 따른 키의 통곗값과 이상치를 분석하려고 한다. 원활한 분석을 위해 데이터를 시각화해 보자.

2. 데이터 수집하기

빅데이터 전처리 과정을 통해 정리한 칼로리-운동 데이터를 사용한다.

3. 데이터 탐색하기

  • ①팝업창의 새 문서 버튼을 클릭한다.
  • ②오른쪽 상단의 실습 파일 보기 버튼( )을 클릭하여 distribution_height.codap의 주소를 복사한다.
  • ③메뉴( )의 ‘가져오기’에서 URL 탭을 선택하여 복사한 주소를 붙여 넣은 뒤 데이터 가져오기 버튼을 클릭한다.

4. 데이터 시각화하기

  • ①그래프를 선택한 뒤, 를 선택한다.
  • ②▼ Box Plot and Normal Curve 항목에서 ‘상자그림’, ‘이상치 보기’를 선택한다.
  • ③▶ Measure of Spread 항목에서 ‘표준편차’를 선택한다.
  • ④▶ Measure of Center 항목에서 ‘평균’, ‘중앙값’을 선택한다.
  • ⑤'측정 레이블 보이기' 항목을 선택한다.
  • ⑥마우스 커서를 그래프에 보이는 이상치 표시(+)에 가져가 값을 확인해 본다.

5. 결과 해석하기

박스 플롯을 보고 그래프의 결과를 해석해 보자.
시각화한 결과를 보면, 남성 키의 평균값은 184.5cm, 중앙값은 184cm임을 알 수 있다. 또한 최솟값은 151cm이고, 최댓값은 217cm이다. 한편, 여성 키의 평균값은 164.4cm이고, 중앙값은 165cm, 최솟값 123cm, 최댓값 197cm임을 알 수 있다.
Tip!
중앙값(Median): 어떤 주어진 값들을 크기의 순서대로 정렬했을 때 가장 중앙에 위치하는 값을 의미한다. 예를 들어, 150, 152, 178 세수의 평균값은 160이고, 중앙값은 152이다. 이처럼 다른 데이터와 다른 극단적이 값이 있을 때는 평균값보다는 중앙값이 통계적으로 더 의미가 있다.
