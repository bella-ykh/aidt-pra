1. 프로그래밍

표의 예시 데이터를 보고 빈칸을 채워 오른쪽 코드를 완성해 보자.
Hint 변수 취미에 문자열 데이터를 저장하는 방법
취미 = '영화 보기'
Hint 변수 키에 실수 데이터를 저장하는 방법
키 = 162.7
Hint 변수 안경_여부에 불형 데이터를 저장하는 방법
안경_여부 = False
Hint 변수 학번에 정수 데이터를 저장하는 방법
학번 = 10516

2. 결과 확인 및 디버깅

  • ①프로그램을 실행하여 [실행 결과 예시]와 같이 결과가 출력되는지 확인해 보자.
  • ②예시와 같이 결과를 출력했다면 ‘채점 하기’ 버튼을 눌러 실습 결과를 확인해 보자.
