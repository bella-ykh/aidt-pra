비밀번호

새로 설정한 우리집 현관문 비밀번호와 이번 달 코드를 바르게 입력해야 집에 들어갈 수 있는 프로그램을 작성해 보자.

1문제 이해 및 분석

우리집에 들어가는 방법 및 절차를 생각해 보자.
  • ①비밀번호를 입력한다.
  • ②이번 달 코드를 입력한다.
  • ③비밀번호와 이번 달 코드가 모두 올바르면 집에 들어간다.
  • ④비밀번호 또는 이번 달 코드가 하나라도 틀리면 집에 들어갈 수 없다.

2알고리즘 설계

①프로그램을 위해 구현해야 할 기능을 생각해 보자.

프로그램을 위해 구현해야 할 기능
달력 모양 키패드
  • 비밀번호를 입력받고, 입력한 값을 비밀번호 변수에 저장한다.
  • 이번 달 코드를 입력받고, 입력한 값을 변수 코드에 저장한다.
  • 비밀번호와 코드가 일치하면 집에 들어간다.(환영 인사 말풍선)
  • 비밀번호 또는 코드가 틀리면 집에 들어갈 수 없다.(접근 불가 말풍선)
Tip
비밀번호는 ‘1234’, 이 달의 코드는 ‘여름’이다.

②프로그램에 필요한 변수를 생각해 보자.

  • 비밀번호: 사용자가 입력한 비밀번호를 저장
  • 코드: 사용자가 입력한 코드를 저장

③프로그램에 필요한 연산자를 생각해 보자.

  • : 사용자가 입력한 비밀번호와 코드가 모두 일치하는지 판단
